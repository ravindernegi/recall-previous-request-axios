
## System Requirement 

* [Nodejs] [http://nodejs.org]
* [Mongodb] [http://mongodb.com]


## Setup

make a clone of this repo after that you need run following commands.

```bash
npm install

```

create a `.env` from `.env.example`  

fill the values as per environment info.



## Run Application
On Local System
```bash
npm run dev

```

On Server
```bash
npm run start

```
